# Theme System

Папка содержит систему тем и цветов приложения.

## Файлы

- `app_theme.dart` - Основная тема приложения (светлая/темная)
- `app_colors.dart` - Цветовая палитра приложения
- `README.md` - Документация темы

## 🎨 Жизнерадостная цветовая схема

## 🎨 Особенности дизайна

### Дружелюбность
- **Мягкие цвета** вместо агрессивных
- **Теплые оттенки** для создания позитивного настроения
- **Высокая контрастность** для лучшей читаемости

### Консистентность цветов
- **Тексты**: Черные в светлой теме, белые в темной
- **Интерактивные элементы**: Синий цвет для кнопок и иконок
- **Поля ввода**: Стильные с закругленными углами и правильными цветами

### Доступность
- **WCAG AA** совместимые цвета
- **Высокий контраст** между текстом и фоном
- **Четкие границы** для элементов управления

### Современность
- **Material Design 3** принципы
- **Плавные переходы** между цветами
- **Консистентная** цветовая схема

## 🚀 Использование

### В коде
```dart
// Использование основных цветов
Container(
  color: AppColors.primary,
  child: Text('Текст', style: TextStyle(color: AppColors.onPrimary)),
)

// Использование состояний
Container(
  color: AppColors.success,
  child: Icon(Icons.check, color: AppColors.onPrimary),
)

// Стильные поля ввода (автоматически применяются)
TextField(
  decoration: InputDecoration(
    labelText: 'Имя',
    hintText: 'Введите ваше имя',
    prefixIcon: Icon(Icons.person),
  ),
)
```

### В теме
```dart
// Автоматическое применение через тему
Theme(
  data: AppTheme.light(),
  child: MyWidget(),
)
```

## 🌙 Темная тема

Темная тема автоматически адаптирует все цвета для комфортного использования в условиях низкой освещенности:

- **Темные фоны** с синеватым оттенком
- **Белые тексты** для лучшей читаемости
- **Синие акценты** для интерактивных элементов
- **Стильные поля ввода** с темными границами
- **Улучшенная** читаемость в темноте

## 🎨 Стильные TextField

### Особенности дизайна:
- **Закругленные углы**: 20px радиус
- **Увеличенные отступы**: 20px горизонтально, 16px вертикально
- **Синие иконки**: prefixIcon и suffixIcon в основном цвете
- **Адаптивные границы**: Серые в светлой теме, темно-серые в темной
- **Фокусное состояние**: Синяя граница при фокусе
