# Verification Feature

Папка содержит функциональность верификации кода подтверждения.

## Файлы

- `verification_screen.dart` - UI экрана верификации
- `verification_viewmodel.dart` - Бизнес-логика экрана верификации

## Функциональность

- Ввод 4-значного кода подтверждения
- Таймер повторной отправки кода
- Валидация кода
- Переход к экрану успеха при правильном коде
- Обработка ошибок

## Архитектура

- **MVVM** паттерн
- **Riverpod** для управления состоянием
- **GoRouter** для навигации
- **Timer** для обратного отсчета
