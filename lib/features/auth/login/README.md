# Login Feature

Папка содержит функциональность входа в аккаунт.

## Файлы

- `login_screen.dart` - UI экрана входа
- `login_viewmodel.dart` - Бизнес-логика экрана входа

## Функциональность

- Ввод номера телефона
- Валидация данных
- Переход к верификации
- Обработка ошибок

## Архитектура

- **MVVM** паттерн
- **Riverpod** для управления состоянием
- **GoRouter** для навигации
