# Registration Feature

Папка содержит функциональность регистрации нового аккаунта.

## Файлы

- `registration_screen.dart` - UI экрана регистрации
- `registration_viewmodel.dart` - Бизнес-логика экрана регистрации

## Функциональность

- Выбор фото профиля
- Ввод имени пользователя
- Ввод номера телефона
- Валидация данных
- Переход к верификации
- Обработка ошибок

## Архитектура

- **MVVM** паттерн
- **Riverpod** для управления состоянием
- **GoRouter** для навигации
- **Image Picker** для выбора фото
