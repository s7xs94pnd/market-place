# Success Feature

Папка содержит функциональность экрана успешной верификации.

## Файлы

- `success_screen.dart` - UI экрана успеха
- `success_viewmodel.dart` - Бизнес-логика экрана успеха

## Функциональность

- Отображение приветственного сообщения
- Персонализированное приветствие с именем
- Разные сообщения для регистрации и входа
- Автоматический переход через 3 секунды
- Кнопка ручного перехода в приложение

## Архитектура

- **MVVM** паттерн
- **Riverpod** для управления состоянием
- **GoRouter** для навигации
- **Timer** для автоматического перехода
