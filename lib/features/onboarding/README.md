# Onboarding Feature

Папка содержит функциональность онбординга (вводного экрана) приложения.

## Файлы

- `onboarding_screen.dart` - UI экрана онбординга
- `onboarding_viewmodel.dart` - Бизнес-логика экрана онбординга

## Функциональность

- Отображение 3 страниц онбординга с анимацией
- PageView с плавными переходами между страницами
- Кнопка "Пропустить" для быстрого перехода к последней странице
- Кнопка "Начать" на последней странице
- Обработка навигации назад (swipe и кнопка)

## Страницы онбординга

1. **Добро пожаловать** - приветствие и краткое описание
2. **Будьте в курсе** - информация об уведомлениях
3. **Готовы стартовать?** - финальная страница с кнопкой "Начать"

## Архитектура

- **MVVM** паттерн
- **Riverpod** для управления состоянием
- **PageController** для управления страницами
- **LocalStorageRepository** для сохранения состояния онбординга
- **NavigationService** для навигации

## Особенности

- **PopScope** для обработки навигации назад
- **Анимации** плавных переходов между страницами
- **Адаптивный дизайн** с фоновым логотипом
- **Автоматическое определение** следующего экрана после завершения
