# Splash Feature

Папка содержит функциональность экрана загрузки приложения.

## Файлы

- `splash_screen.dart` - UI экрана загрузки
- `splash_viewmodel.dart` - Бизнес-логика экрана загрузки

## Функциональность

- Отображение логотипа приложения
- Индикатор загрузки
- Определение следующего экрана на основе сохраненных данных
- Автоматическая навигация после инициализации
- Обработка ошибок инициализации

## Логика определения маршрута

Экран определяет следующий экран на основе:
- **Onboarding** - если онбординг не пройден
- **Auth** - если онбординг пройден, но нет токена
- **Home (User)** - если есть валидный токен
- **Home (Guest)** - режим гостя

## Архитектура

- **MVVM** паттерн
- **Riverpod** для управления состоянием
- **UseCase** для бизнес-логики определения маршрута
- **LocalStorageRepository** для работы с локальными данными
- **NavigationService** для навигации

## Особенности

- Минимальная задержка 1 секунда для показа анимации
- Автоматическое заполнение тестовых данных для разработки
- Graceful обработка ошибок с fallback на экран аутентификации
